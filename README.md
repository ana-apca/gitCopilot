# 🤖 GitHub Copilot — Treinamento

Repositório de exercícios práticos do treinamento de desenvolvimento assistido por IA com **GitHub Copilot**.

## Sobre o Projeto

Exercícios desenvolvidos durante o curso "Code with GitHub Copilot", explorando funcionalidades de autocompletar, geração de código e testes unitários com Copilot no VS Code e Codespaces.

## Conteúdo

### Prime Number Checker
Aplicação console em C# para identificar números primos, com testes unitários.

- `PrimeNumberChecker.cs` — Classe com métodos para identificar primos de 1 a 100
- `PrimeNumberCheckerTests.cs` — Testes unitários com xUnit

## Tecnologias

- **Linguagem:** C#
- **Framework:** .NET Core
- **Testes:** xUnit
- **IA:** GitHub Copilot
- **Ambiente:** GitHub Codespaces

## Como Executar

1. Clone o repositório
2. Abra no VS Code com Copilot habilitado
3. Execute os testes: `dotnet test`
